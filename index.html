Project Highlights

AEROSON: SOUND AIR MONITORING SYSTEM

AEROSON is a smart IoT-based system for real-time air and sound pollution monitoring at the community level. It measures particulate matter (PM2.5 and PM10), harmful gases like carbon monoxide and nitrogen dioxide, and environmental noise using sensors controlled by an ESP32 microcontroller. Data are sent to a cloud server to compute Air Quality Index values, analyze trends, and generate visual reports via web and mobile apps, offering dashboards, alerts, and predictive analysis for informed decision-making and public health protection.

MJ BEAUTY SHOP: POINT OF SALE SYSTEM

The Point of Sale and Inventory Management System for MJ Beauty is a web-based application built using PHP, with MySQL as its backend database. It provides a secure, role-based platform for administrators, staff, and customers to manage inventory, process sales, and place orders online. The system features real-time stock tracking, an intuitive product catalog, and automated reporting, streamlining operations for the cosmetics business.
